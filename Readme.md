## About

Installs:

* required system packages
* [InfluxDB](http://influxdb.com/) 0.9

## Requirements

* Vagrant >=1.7
* VirtualBox >=4.0

### For UNIX-based systems

* [Ansible](http://docs.ansible.com/intro_installation.html) >=1.8 

Ansible roles:

```shell
ansible-galaxy install palkan.influxdb
```
